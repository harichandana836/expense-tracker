# 💰 Expense Tracker
A simple and responsive **Expense Tracker Web App** built using **HTML, CSS, and JavaScript**.
This application allows users to manage their daily expenses with features like adding, editing, deleting, filtering expenses, and visualizing spending using charts.

## 🚀 Live Demo
👉 https://harichandana836.github.io/expense-tracker

## ✨ Features
* Add new expenses
* Edit existing expenses
* Delete expenses
* Category filter
* Total expense calculation
* Transaction counter
* Pie chart visualization of expenses
* Persistent storage using Local Storage
* Responsive design (Mobile Friendly)

## 🛠 Technologies Used
* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Local Storage API
* Chart.js
* GitHub Pages (Deployment)

## 📊 Functionality
This application allows users to:
